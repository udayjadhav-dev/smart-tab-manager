# Smart Tab Manager 🗂️

A Chrome Extension that automatically groups your open tabs by category with one click!

## Features
- One-click tab grouping
- Auto-detects categories — Work, Shopping, YouTube, News, Social
- Color-coded tab groups
- Lightweight and fast — works 100% locally

## Categories
| Category | Sites |
|----------|-------|
| Work | GitHub, Notion, Google Docs, Slack |
| Shopping | Amazon, Flipkart, Myntra |
| YouTube | YouTube |
| News | NDTV, BBC, Times of India |
| Social | Twitter/X, Instagram, LinkedIn |

## Installation
1. Clone this repo
2. Run `npm install`
3. Run `npm run build`
4. Open Chrome → `chrome://extensions`
5. Enable Developer Mode
6. Click Load unpacked → Select `dist` folder

## Tech Stack
- React 18
- Vite 5
- Chrome Extensions API (Manifest V3)
